# Realty - Инвестиционный аналитик ИС (.NET / ASP.NET Core Web API)

Учебный групповой проект: Мини-информационная система для оценки экономической эффективности внедрения информационных систем (TCO, RIO, Payback Perlod, анализ чувствительности, сравнение локального и облачного сценариев).

Этот вариант использует **C# и ASP.NET Core Web (.NET 8.0)**

## Быстрый старт

'''bash
dotnet restore
dotnet build
dotnet run --project srt/api/api.cspro
'''
